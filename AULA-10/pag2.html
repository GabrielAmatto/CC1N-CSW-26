Capítulo 3 (External)

CSS (Cascading Style Sheets) é uma linguagem usada para estilizar páginas web, controlando cores, fontes, espaçamentos e layouts. Ele funciona em conjunto com o HTML, permitindo separar o conteúdo da apresentação visual. Com CSS, é possível criar designs responsivos que se adaptam a diferentes dispositivos e tamanhos de tela. Além disso, facilita a manutenção do site, pois alterações de estilo podem ser feitas de forma centralizada.
